## What is Demo Image Classification Part 1

Taipy is a Python library for creating Business Applications. More information on our
[website](https://www.taipy.io).

[Demo Image Classification Part 1](https://github.com/Avaiga/demo-image-classification) is a front end application built with Taipy GUI to transform classifier into fully interactive application


### Demo Type
**Level**:  Intermediate

**Topic**: Taipy GUI

**Component**: Layouts; Visual Elements (chart, button, date input, table, image)

### Description: 

In this video, we are building a front end application for a tensorflow image classifier
### Introduction:

This video is designed to be the first of a 2-part tutorial about using Taipy GUI & Taipy Core with tensorflow
